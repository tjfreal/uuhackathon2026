# UU Hackathon 2026

This repository is a single hackathon submission that packages two related example builds plus the planning material that surrounded them.

It is not organized like a polished open-source tool release. The point of this repo is to show what was explored, what actually got built, and what those builds suggest for the next round of work.

## What I Went In With

- a prompt to generate ideas for second-brain organization tools
- a couple of in-progress projects
- a list of possible projects to explore
- one personal benchmark project to use as a stress test

## What I Ended Up Making

- [julia-stl](julia-stl): an interesting take on the personal benchmark project, built around turning Julia-set structure into a printable vase workflow
- [decksmith](decksmith): a potentially useful local Keynote / PowerPoint generator built on AppleScript and simple text-based deck packages

## What Else Is Here

- roadmaps and learnings for both projects
- presentation framing documents that explain how the projects changed as they were built
- deck packages and project stubs that capture several adjacent ideas explored during the hackathon, even when they were not taken to a finished state

## Notes On Scope

These projects are included here as hackathon examples, not as production-ready standalone tools.

- `julia-stl` is the stronger artifact build
- `decksmith` is the stronger reusable workflow idea
- both are still early
- the surrounding docs are part of the submission because the process and reframing are a real part of what was accomplished
