# Presentation

Decksmith is a local-first tool for turning project thinking into editable presentations.

It sits between markdown planning documents and slide software:

- write the ideas in plain text
- condense them into slide rows with notes
- optionally attach images where they matter
- generate a deck locally in Keynote
- export to `.pptx`
- refine by hand

The most important thing about Decksmith now is that it has crossed out of proposal territory. It has already been used on multiple real decks, and the current repo structure makes that pattern legible instead of accidental.
