# Narrative Plan

## Objective

Explain Throughline as both a practical hackathon build and a broader argument about retrieval granularity in LLM-era knowledge systems.

## Narrative Arc

1. Name the project and the problem space.
2. State the core claim about retrieval granularity.
3. Explain the system behavior in operational terms.
4. Show why it is feasible now.
5. Outline the build plan.
6. Close on the larger significance.
