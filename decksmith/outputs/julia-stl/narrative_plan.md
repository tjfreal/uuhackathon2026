# Narrative Plan

## Objective

Explain `julia-stl` as a project that discovered a method for guided aesthetic tuning rather than merely producing a single fractal object.

## Narrative Arc

1. Start with the original fractal-to-STL ambition.
2. Show why the naive path failed.
3. Explain the conceptual shift.
4. Show the geometric and tooling breakthroughs.
5. End on the current refinement frontier.
