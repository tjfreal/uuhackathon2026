# Narrative Plan

## Objective

Present the hackathon project set as a coherent portfolio of local, bounded, useful systems rather than a disconnected list of demos.

## Narrative Arc

1. Introduce the shared build philosophy.
2. Show the project set as variations on that philosophy.
3. Use Throughline as the anchor example.
4. Explain the privacy and demo boundary.
5. Close on the larger lesson about trustworthy, durable tools.
